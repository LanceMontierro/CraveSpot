# CraveSpot

CraveSpot is a MERN stack application for managing dishes, orders, and users.

## Features

- User authentication and management
- CRUD operations for dishes and orders
- Responsive design

## Technologies && Tools Used

- MongoDB
- Express.js
- React
- Node.js
- Vite
- Tailwind Css
- Material Ui
- Toastify
- Geolocation API
- Clerk
- react-starfield
